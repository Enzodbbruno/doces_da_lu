🚀 Como usar este painel

  • Produtos: Edite o nome, preço, descrição e peso de cada ovo. Clique em Salvar.
  • Disponibilidade: Use o botão "Disponível/Pausado" para ativar ou desativar produtos.
  • Configurações: Configure sua chave PIX, WhatsApp e chave do Mercado Pago.
  • Imagens: Para trocar as fotos, faça upload de uma nova imagem no card do produto.
  • Mercado Pago: Insira sua chave pública MP nas Configurações para pagamento real.
